

This is a game to teach geography in multiple languages (EN and FR)

# Features

## From the world map
- Reply each country the highlighted one is from a list of options
- From a country name, point it on the map

## With flags
- Name the country for which this flag is from (from a list)
- Choose the correct flag of the proposed country

## For region
- For all game modes, I can also choose from which region (and sub-region) I want to play:

- Europe
    - East europe
    - North 
    - ...
- Africa
    - ...
- Asia
- ...


## Teaching
- The app should track which country/flags/.. are not correct so that it can provide dedicated learning sessions to help teach the one that are often wrong.
Example: During game, if the users misidentify Bulgaria and Romania multiple times, there should be a mode available to train on this country that were guessed wrong
- The app should keep the play history to track how the user progress, and how much time it took to complete the exercise

## Tech
- It should be a SPA running in the web
- You can use a backend if needed. If you need a DB use something really lightweight
- Keep the requirements low. Only use what's strictly needed, without reinventing the wheel
